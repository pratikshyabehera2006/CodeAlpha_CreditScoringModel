# 💳 Credit Scoring Model - CodeAlpha Internship

## 📌 Objective
This project predicts whether a person is creditworthy using Machine Learning.

## 📊 Dataset
German Credit Dataset (UCI ML Repository)

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## 🤖 Models Used
- Logistic Regression
- Random Forest

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## 🌐 Web App
A Streamlit web application is created for real-time prediction.
